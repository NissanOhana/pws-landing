What You'll Learn

Three pillars of
web engineering

The course covers the full spectrum of web development — not just coding, but deep understanding of the systems, protocols, and architecture behind every web application.

Frontend Engineering

Understand how browsers really work — from parsing HTML to the critical rendering path. Master async programming, build reactive UIs with React, and learn to optimize performance.

HTML & DOM CSS JavaScript TypeScript React Async Event Loop Browser API

Backend Development

Build secure, scalable web servers with Node.js. Design RESTful APIs, implement authentication with JWT, connect to databases, and handle real-world security threats.

Node.js REST APIs HTTP/HTTPS Authentication MongoDB Cookies CORS Web Security

System Architecture

Think beyond code. Design microservice architectures, implement event-driven systems with message brokers, and understand distributed data challenges like CAP theorem and ACID.

Microservices Message Brokers Event-Driven ACID CAP Theorem Docker CI/CD System Design
The Curriculum

11 weeks from zero to
full-stack architect

Each week builds on the last. Lectures blend theory, live coding, and hands-on practice into a single session.

Frontend
Week 01
Web Basics & The Journey of a Request
How does clicking "Like" traverse the entire internet? Explore the OSI model, TCP/UDP, DNS, and HTTP from first principles.
OSI Model TCP/UDP DNS HTTP URLs
Week 02
Frontend Foundations
Build your first web page. Understand how browsers construct the DOM, apply styles, and execute JavaScript. Browser APIs and event handling.
HTML CSS JavaScript DOM Events Browser API
Week 03
Async Programming & Performance
Deep dive into the JavaScript engine and event loop. Understand the critical rendering path and how to write non-blocking, efficient code.
Event Loop Promises async/await Critical Rendering Path JS Engine
Week 04
Modern Frontend & React
Enter the modern frontend ecosystem. NPM, module bundlers, TypeScript for type safety, and building reactive user interfaces with React.
NPM TypeScript React Components State Hooks
Backend
Week 05
Backend Fundamentals & Node.js
Cross the bridge from frontend to backend. How web servers process requests, REST API design principles, and building your first server with Node.js.
Web Servers REST Node.js Express API Design
Week 06
Backend Core: Cookies, CORS & Databases
Connect your server to the real world. Session management with cookies, cross-origin resource sharing, and integrating MongoDB for persistent data.
Cookies CORS MongoDB Sessions Data Modeling
Week 07
Authentication & Web Security
Secure your applications. TLS/HTTPS, JWT-based authentication, password hashing, and defending against XSS, CSRF, and injection attacks.
TLS/HTTPS JWT Authorization XSS CSRF SQL Injection
System Architecture
Week 08
Microservices & Event-Driven Architecture
Scale beyond monoliths. Design microservice systems, implement event-driven communication, and use message brokers like RabbitMQ for async workflows.
Microservices Event-Driven RabbitMQ Pub/Sub API Gateway
Week 09
Managing Data at Scale
Understand distributed data challenges. Transactions, ACID properties, the CAP theorem, and strategies for data consistency across microservices.
Transactions ACID CAP Theorem Eventual Consistency Saga Pattern
Week 10
Large-Scale Systems
Deploy and operate real systems. Caching strategies, CI/CD pipelines, containerization with Docker and Kubernetes, and system design exercises.
Caching CI/CD Docker Kubernetes System Design
Week 11
Advanced Topics & Design Practice
Push the boundaries. Web Workers, WebSockets for real-time, gRPC for efficient services, and hands-on system design sessions for production-grade architectures.
Web Workers WebSockets gRPC Scaling Design Practice

Teaching Approach

Learning by doing

This is a practical engineering course. Theory, live coding, and hands-on exercises come together in every session.

Live Coding

Concepts are taught through real-time coding in lectures. See how systems are built, debug problems live, and understand the thought process.

Workshops

Three guided workshops — React, APIs & Databases, and Full-Stack — provide structured practice with hands-on coding exercises.

Theory Meets Practice

Every theoretical concept connects to a real-world implementation. OSI model to HTTP requests. CAP theorem to your database design choices.
